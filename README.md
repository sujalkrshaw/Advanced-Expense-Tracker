# 💰 Advanced Expense Tracker

## 📌 Overview

The Advanced Expense Tracker is a data-driven financial analytics application built using Python. It allows users to track, analyze, and predict expenses using Data Science and Machine Learning.

This project simulates real-world FinTech systems used by companies like Google Pay, CRED, and Razorpay.

---

## 🎯 Objective

* Track and manage expenses efficiently
* Analyze spending patterns
* Visualize financial data
* Predict future expenses using Machine Learning

---

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Matplotlib, Streamlit
* SQLite (Database)
* Scikit-learn (Machine Learning)

---

## 🔄 Workflow

```text
User Input / Synthetic Data
        ↓
Data Storage (SQLite / CSV)
        ↓
Data Cleaning (Pandas)
        ↓
Data Processing (Grouping, Aggregation)
        ↓
Data Analysis
        ↓
Visualization (Charts)
        ↓
Insights & Reports
```

---

## 📊 Features

* 📌 Category-wise expense analysis
* 📈 Monthly spending trends
* 🔮 Expense prediction using Linear Regression
* 📊 Interactive dashboard using Streamlit
* 🧾 Synthetic dataset generation

## ▶️ How to Run

### 1. Install dependencies

```bash
pip install pandas numpy matplotlib seaborn scikit-learn streamlit joblib
```

### 2. Run data pipeline

```bash
python main.py
```

### 3. Run dashboard

```bash
streamlit run app/app.py
```
